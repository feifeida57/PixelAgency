# 🗺️ Roadmap

PixelAgency 的发展路线图。

## ✅ MVP（已完成）

核心功能已实现：

- [x] 215 位 AI 智能体定义（17 个部门）
- [x] 像素风虚拟办公室（Canvas 2D）
- [x] 7 种智能体状态（工作、游戏、睡觉、咖啡、聊天、发呆、走动）
- [x] 部门选择器
- [x] LLM 秘书系统提示词
- [x] 智能体索引生成脚本

## ✅ v0.1（已完成）

增强交互体验：

- [x] WebSocket 实时状态同步（[#1](https://github.com/feifeida57/PixelAgency/issues/1)）
- [x] 更丰富的行为动画（[#2](https://github.com/feifeida57/PixelAgency/issues/2)）
- [x] 智能体互动系统（[#3](https://github.com/feifeida57/PixelAgency/issues/3)）
- [x] 可独立运行的 Demo 页面
- [x] 截图/GIF 展示

## 📋 v0.2（进行中）

扩展功能：

- [x] 智能体搜索功能
- [x] 任务历史记录
- [ ] 智能体性能统计
- [ ] 自定义办公室布局
- [x] 主题切换（深色/浅色）
- [ ] 音效系统

## 🔮 v0.3（远期）

高级特性：

- [ ] 多办公室场景（会议室、休息室、健身房）
- [ ] 智能体技能树系统
- [ ] 成就系统
- [ ] 办公室装修系统
- [ ] 智能体社交网络
- [ ] 实时协作（多人同时查看）

## 🎯 长期愿景

让 PixelAgency 成为：
1. **AI 团队可视化标准** — 展示多智能体协作的最佳实践
2. **教育工具** — 帮助理解 AI agent 系统的工作原理
3. **娱乐产品** — 让 AI 不再冰冷，而是有"生命"的办公室
